# Homework

## Описание
Проект для изучения основ объектно-ориентированного программирования на Python.
Реализованы классы `Product` и `Category` для интернет-магазина.

## Функциональность
- Класс `Product` с атрибутами: name, description, price, quantity
- Класс `Category` с атрибутами: name, description, products
- Классы-наследники: Smartphone, LawnGrass
- Подсчёт общей стоимости товаров на складе (магический метод __add__)
- Проверка типов при сложении товаров

### Абстрактный класс BaseProduct
- Базовый абстрактный класс для всех продуктов
- Содержит абстрактные методы `__str__` и `__add__`
- Обеспечивает единый интерфейс для всех товаров

### Миксин LogMixin
- Автоматически логирует создание объектов
- Выводит в консоль: класс объекта и переданные параметры
- Добавлен в цепочку наследования класса `Product`

### Обработка исключений
- При создании товара с quantity=0 выбрасывается `ValueError`
- Сообщение: "Товар с нулевым количеством не может быть добавлен"

### Метод middle_price() в классе Category
- Вычисляет среднюю цену всех товаров в категории
- При пустой категории возвращает 0 (без ошибки)
